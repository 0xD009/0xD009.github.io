# 0x7 flag 自动机

flag{Y0u_rea1ly_kn0w_Win32API_89ab91ac0c}
菜鸟的老本行
在这一步修改 zeroflag 绕过反调试

只要进入这个 0x111，114514 的分支就能拿到 flag 了

注意看 0x1BF52，容易看出这就是十六进制的 114514，做一个比较然后做一个跳转，前面有一个 jmp 跳转到生成窗口的地方，set eip 跳过它，然后改两次 zf 到生成 flag 文件的地方，继续执行就拿到 flag 了
