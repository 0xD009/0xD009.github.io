# 2. buu helloword

下载附件得到一个 APK 文件，之前稍微学了一点点 Java 和安卓开发，又去看了一下 APK 逆向的流程，大概能做。

D:\APK_reverse>java -jar jd-gui-1.6.6.jar

是明显的迷宫题， v3 的前 24 位就是迷宫地形，后面四位存 y 坐标，再往后紧接着声明的是 x ，即 x 坐标，所以最后一个 for循环 是检验玩家是不是走出迷宫范围了，也可知迷宫是 5X5 地形。
考究一下最后成功判定中 41 的含义，实际上指 v7 和 v3 首地址的差值（注意 x , command , i 各占 4 字节），属于是迷惑手段了。
