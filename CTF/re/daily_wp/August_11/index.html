用 JavaScript 写成的，百度了解一下用到的语法，大意是把前 13 个字母移位到后 13 位，后 13 位移位到前 13 位，根据目的写脚本：

a = "PyvragFvqrYbtvafNerRnfl@syner-ba.pbz"
b = ''
for i in a:
    if ('a' <= i <= 'z'):
        if ord(i) <= 122 - 13:
            b += chr(ord(i) + 13)
        else:
            b += chr(ord(i) - 13)
    elif ('A' <= i <= 'Z'):
        if ord(i) <= 90 - 13:
            b += chr(ord(i) + 13)
        else:
            b += chr(ord(i) - 13)
    else:
        b += i
print(b)

解得 ClientSideLoginsAreEasy@flare-on.com ，整理格式后提交，解毕。
